# Airifica Web — AIR3 Agent Interface

**Airifica Web** is the production frontend for **AIR3 Agent** — an AI-powered trading assistant built on top of [AIRewardrop](https://airewardrop.xyz) and the [AIR3 ecosystem](https://airewardrop.gitbook.io/air3).

It combines a real-time 3D avatar, conversational AI (powered by [elizaOS](https://elizaos.ai)), live on-chain market data, and direct integration with the **Pacifica DEX** to give users a fully interactive, voice-capable AI trading companion directly in the browser.

---

## What AIR3 Agent Does

| Capability | Description |
|---|---|
| **AI Conversation** | Chat with Bairbi, the AIR3 trading agent, powered by elizaOS and DeepSeek LLM |
| **3D Avatar** | Real-time VRM character with facial expressions, lip-sync, and gesture animations |
| **Market Context** | Live price, candle chart, funding rate, open interest — injected into every message |
| **Trade Proposals** | Structured trade setups (entry, TP, SL, leverage, confidence score) generated by the AI |
| **Trade Execution** | One-click order submission through the Pacifica DEX builder integration |
| **Voice Output** | Text-to-speech narration of AI responses with phoneme-accurate lip-sync |
| **Voice Input** | Browser microphone with Voice Activity Detection (VAD) for hands-free interaction |
| **Solana Auth** | Wallet-based authentication via Phantom, Backpack, or any injected Solana provider |
| **Iframe Embed** | Embeddable in partner sites via `postMessage` bootstrap protocol |

## Tech Stack at a Glance

```
Browser (Vue 3 SPA)
  └─ Three.js + @pixiv/three-vrm   → 3D avatar rendering
  └─ wlipsync                      → Phoneme-to-viseme lip sync
  └─ Web Audio API                 → PCM audio streaming
  └─ VAD Web Worker                → Voice activity detection

Port-Bridge (Node.js HTTP Gateway — port 5173)
  └─ Serves dist/ static files
  └─ Proxies /api/* → elizaOS (port 4040)
  └─ Proxies /api/tts* → TTS proxy (port 4041)

ElizaOS / AIR3 Backend (port 4040)
  └─ Express.js + elizaOS runtime
  └─ DeepSeek LLM
  └─ Pacifica API integration
  └─ Solana wallet verification

TTS Proxy (port 4041)
  └─ Forwards to Kokoro / OpenAI-compatible TTS backend
  └─ Supports captioned speech & phoneme extraction
```
